# KDJ Cream Website

This is a premium, multi-page website built for "KDJ Cream" using HTML, Tailwind CSS, and JavaScript.

## Setup
No installation required! This is a static website.
Simply open `index.html` in your web browser to view the site.

## Structure
- `index.html`: The main landing page with hero section, features, and testimonials.
- `products.html`: Lists the KDJ Cream products with a "Buy Now" option.
- `about.html`: Information about the brand, story, and ingredients.
- `contact.html`: A contact form and location details.
- `script.js`: Handles interactions like the mobile menu and scroll effects.

## Technologies
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Font Awesome (for icons)
- Google Fonts (Playfair Display & Poppins)
